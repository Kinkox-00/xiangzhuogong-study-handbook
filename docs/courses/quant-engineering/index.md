# 量化工程分析

本课程复习站先按现有资料拆成四个方向。当前已补充图论、动态系统最优化控制、拉格朗日方程与自行车/轮式机器人运动学，后续可以继续按自测题增加例题。

## 复习模块

| 模块 | 当前状态 | 主要内容 |
|---|---|---|
| [图论基础](./graph-theory.md) | 已补充 | 图的表示、连通性、Euler/Hamilton、最短路、网络流、最小生成树 |
| [动态系统最优化控制方法](./optimal-control.md) | 详版 | 变分法、Euler 方程、Pontryagin 最小值原理、LQR/Riccati |
| [拉格朗日力学](./lagrange.md) | 已补充 | 约束、虚位移、理想约束、达朗伯原理、拉格朗日方程 |
| [自行车与轮式机器人运动学](./kinematics.md) | 已补充 | 2D bicycle model、纯滚动、无侧滑、转弯半径、轮式机器人移动性 |

## 推荐复习顺序

1. 先看图论概念和算法，拿稳选择题与流程题。
2. 再看最优化控制方法，掌握题型入口和公式模板。
3. 最后补拉格朗日力学与运动学简答题。

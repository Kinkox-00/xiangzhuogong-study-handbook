## 真题复习结论

近年卷面结构比较稳定：总分 100 分，考试时间 120 分钟。复习收益最高的是汇编与 C 互推，其次是 Cache / TLB / 页表综合题，再然后是数值表示、链接、信号与程序优化。

三天冲刺时，不建议平均刷所有旧卷。更好的顺序是：

1. 精做 2025、2024、2023 三套近年卷。
2. 旧卷只抽同题型：浮点题、汇编题、Cache 题、链接题。
3. 每道错题记录“题型、卡住点、正确步骤、下次识别信号”。

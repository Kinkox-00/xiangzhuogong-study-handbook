## 可以反馈什么

| 类型 | 适合提交的内容 |
| --- | --- |
| 错误反馈 | 公式错误、题解错误、算法步骤不严谨、排版异常、链接失效 |
| 缺漏补充 | 缺少知识点、缺少高频题型、课程章节不完整、例题覆盖不足 |
| 内容投稿 | 更好的讲解、例题解析、知识清单、图片资料、教材出处或复习经验 |

## 投稿格式

复制下面这段到邮件里即可：

```text
课程名称：
页面链接或章节名称：
问题类型：
具体说明：
建议修改内容：
可选署名：
```

## 提交建议

- 如果是公式或推导问题，请尽量说明“原页面哪里错了”和“建议改成什么”。
- 如果是题目解析投稿，请附上题目来源、关键步骤和最终答案。
- 如果是图片、课件或教材资料，请尽量说明来源，避免版权不清的内容直接公开。
- 如果只是提醒缺漏，也可以只写课程名和缺少的知识点。
